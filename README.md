# Go Dev Env Template

# Description

This is template project for development with golang(1.20).
enviroment build on docker container.
keep clean your local enviroment.

# Requirement

* VSCode
* Docker(Desktop)
* Dev Containers

# Usage

1. check out repo.
2. open directory on VSCode.
3. Reopen in container

# Test Run

```bash
go run cmd/main.go
```
